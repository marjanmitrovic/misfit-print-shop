Statička prodavnica majica na srpskom i češkom jeziku. Radi bez instalacije i bez servera.

## Pokretanje

Otvorite `index.html` u pregledaču ili postavite ceo folder na GitHub Pages, Render Static Site, Netlify ili drugi statički hosting.
